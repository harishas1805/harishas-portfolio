## **1. Preparation**
- **Tool:** Use a screen recorder like **OBS Studio** (free), **Loom** (browser extension), or **Windows Key + Alt + R**.

- **Theme:** Start in **Dark Mode** (since it's the default and looks premium).

## **4. How to Add Voice?**

### **Option A: Record Yourself (Best for Recruiters)**
1.  Open **Voice Recorder** on your phone.
2.  Find a quiet room.
3.  Read the script above while watching your screen recording.
4.  Sync the audio with the video in any video editor (CapCut, Clipchamp).

### **Option B: Use AI Text-to-Speech**
If you want a polished "student" voice without recording:
1.  Go to **ElevenLabs.io** (Free tier available).
2.  Select a voice like "Adam" or "Josh" (American/British young professional).
3.  Paste the script above.
4.  Download the MP3 and add it to your video.
